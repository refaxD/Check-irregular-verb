# Check-irregular-verb
## Телеграм-бот для интересного и лёгкого изучения неправильных глаголов

> **Суть проекта**
> Его основная суть заключается в важной психологической реакции человека на буллинг от незнакомого лица во время обучения. Появляется желание доказать, на что ты способен, при этом не испытывая реальной обиды, ведь нельзя обидеться на кусок кода
>
> *Короче, это просто забавно и эффективно*

## Что умеет этот кусок ~~говна~~ кода?
- **Кидает в тебя глаголами** (чтоб не втыкал)
- **Унижает за ошибки** (чтоб не втыкал!)
- **Сухо хвалит**, если ты вдруг ответил правильно (чтоб не втыкал!!!)


## Способ установки

### 1. Клонируй этот шедевр и кдшнись в корень
```bash
git clone https://github.com/refaxD/Check-irregular-verb.git
cd Check-irregular-verb
```

### 2. Создай виртуалку (опциональненько)

# Windows (линуксоиды и макосники, терпите)
python -m venv .venv
.venv\Scripts\activate

### 3. Установка зависимостей

```bash
pip install -r requirements.txt```

### 4. В .env впиши токен из ботфатхера
